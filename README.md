# unit-converter-python
A simple Python terminal-based Unit Converter that allows users to convert between various units of length, weight, time, and volume.
The user selects the type of conversion from a menu, inputs the value, and the program returns the converted result.

Features :
Length conversions:

Kilometers ↔ Meters

Meters ↔ Centimeters

Weight conversions:

Kilograms ↔ Grams

Time conversions:

Hours ↔ Minutes

Volume conversions:

Liters ↔ Milliliters

Simple menu-driven terminal interface

Easy to extend for more unit conversions

No external libraries required
